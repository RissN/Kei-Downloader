# Background Image

Taruh gambar background di folder ini dengan nama **`bg.jpg`**.

Rekomendasi:
- Resolusi: 1920×1080 atau lebih tinggi
- Tema: Pemandangan gunung Jepang, langit cerah, sakura
- Format: JPG (untuk ukuran file lebih kecil)

App akan otomatis menggunakan gambar ini sebagai background.
Jika tidak ada file, app menggunakan gradient biru langit sebagai fallback.
